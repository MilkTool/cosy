
# Cosy

Cosy is a small and simple programming language. It has an multi-platform interpreter and trans-compile to JavaScript.

```javascript
print 'hello world'
```

```javascript
var max = 3
for i 0..max {
    print 'loop #' + i
}
```

## High-level features

* Familiar syntax
* Lambda functions (anonymous functions)
* Compile-time validation
* (...)
